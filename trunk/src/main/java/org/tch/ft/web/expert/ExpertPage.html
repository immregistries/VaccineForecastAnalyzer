As a registered user of the TCH Immunization System you are considered an expert, whether your expertise is more closely related to immunization practice or software development. As an expert, you may be associated with one or more task groups.

Task Group Assigned

A task group is a set of experts that work together to create a consistent set of test cases and recommendations. If you wish to become part of an existing task group, you will need to contact a member of that task group and request that they add you as a member. If your task group is currently not registered please contact TCH to create a new task group.
