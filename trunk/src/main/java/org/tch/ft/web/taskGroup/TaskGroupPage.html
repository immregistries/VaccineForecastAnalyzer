Task Group

A task group consists of immunization and software experts who work together to configure, deploy, and test immunization forecast systems. The experts of a task group use this website to organize test cases into test panels, give expert opinion on expected results, and verify software meets these expectations.

Task Group Visibility

All task groups are visible to all users of this system. This includes the test panels, test cases, expected results, actual results and comments that experts in this task group make.

Joining a Task Group

To join a task group, you must ask one of the current members to add you. Experts who belong to the task group can add any register expert.

Select

You can select and view any task group within this testing system. If you need to register a new Task Group that is not listed yet, please contact a member of the TCH staff to have it added.

Test Panels

Every Task Group can define one or more Test Panels. A Test Panel is a a collection of test cases that are used for verification.

Test Panels Defined by Task Group

Experts

System users can be assigned as experts on one or more Task Groups. Any Expert or Admin member of a Task Group can add and remove any expert to the group. Expert and Admin members of a Task Group can set expectations to test cases that become the official expectation of the Task Group. Other Experts, who are either View members or not assigned as members may view the associated test panels, test cases, and comments, but if they record expectations they will be displayed as their personal expectations and will not be registered under the Task Group.
